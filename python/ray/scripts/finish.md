*In conclusion...*
<br> 
The Ray TUI is a lightweight tool for observing your Ray cluster.

<br>
It supports more than a dozen features:

* cluster information
* autoscaler information
* tasks / actors list
* viewing nodes information
* attaching to nodes
* objects memory 
* scrolling
* node logs
* actor logs
* cluster mode (ray htop yaml)
* etc
